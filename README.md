# المدير المالي المتكامل

تطبيق ويب لإدارة المصروفات والدخل الشهري، مع صفحة للديون، كشف تفصيلي، وتصدير PDF.

## الميزات
- إضافة دخل ومصروف مع الفئة والوصف والتاريخ التلقائي
- عرض العمليات في جدول
- ملخص إجمالي الدخل والمصروف والرصيد
- صفحة للديون مع نوع الدين وملخص إجمالي
- تصدير PDF للعمليات والديون
- تخزين البيانات في LocalStorage
- تصميم متجاوب للأجهزة المحمولة
